Design Principles

Hick's Law

HelloFresh.com

hellofresh.com/menus

By only having a few items to choose from makes it easier for the user to make a choice. This site only gives a few options at a time so that it doesn't overwhelm the user and allows them to be able to make a decision before giving up.

Fitt's Law

PHI Air Medical

phiairmedical.com

This page uses Hick's Law because there are very few items to click or interact with. The website design makes it easy to know where to click next. On a seperate topic, the alignment doesn't match up and makes the site look messy and amature.

Rule of Thirds

Garmin

garmin.com

I noticed that there wasn't a picture on the Garmin site that the item was directly centered. All items were off set to the left or the right. This allowed the user to be able to take in more of the picture overall and not focus just on the item being sold.
